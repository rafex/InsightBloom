# SPEC.md

## Resumen

Definir la app web del PoC para portal de entrada, nubes de dudas y temas,
timeline por palabra y dashboard de moderacion.

## Incluye

- Estructura `pages`, `features`, `components`, `services`.
- Rutas publicas y privadas.
- Integracion con auth y APIs backend.
- Uso de D3.js, Vue Router, animate.css y Pug.

## Requisitos clave

- Deben existir vistas separadas de dudas y temas.
- El dashboard debe soportar paginacion.
- El frontend obtiene `deviceFingerprint` con ThumbmarkJS.
